# Isha Upanishad - Verse-by-Verse Analysis

This directory contains the granular, verse-by-verse analysis of the Isha Upanishad. Each file corresponds to a single verse and includes:

- Sanskrit text (Devanagari and IAST)
- English translation
- Word-by-word grammatical analysis
- Summary of major commentaries (Shankara, Ramanuja, etc.)
- Philosophical significance
- Practical application

This deep dive serves as the pilot for the "Blue Belt" progression of the Vedic Mastery Study.
